# 实时涂鸦协作板

## 网站简介
实时涂鸦协作板是一个在线的多人协作绘图平台，允许多个用户同时在同一个画布上进行涂鸦和创作。该项目通过多个 AI 模型的协作完成，旨在展示 AI 在复杂项目中的协同工作能力。

## 页面与功能
本网站包含 8 个页面，主要功能包括：
- 首页：介绍网站的功能和特点，提供进入画板的入口。
- 画板：核心功能页面，支持多人实时协作涂鸦。
- 登录/注册：用户身份验证页面，支持用户注册和登录。
- 用户中心：展示用户信息，管理用户创作的作品。
- 作品展示：展示用户创作的作品集。
- 帮助中心：提供使用网站的帮助和指南。
- 关于我们：介绍网站的开发团队和背景。
- 404 页面：处理访问不存在页面的错误。

## 多 AI 协作与验收
本项目由多个 AI 模型协作完成，每个模型负责不同的任务，如：
- 前端开发 AI：负责网站的页面设计和交互逻辑实现。
- 后端开发 AI：负责服务器端的逻辑处理和数据存储。
- 协作功能 AI：实现多人实时协作涂鸦的核心功能。
- 测试 AI：进行网站的功能测试和性能优化。

在项目开发过程中，通过定义清晰的接口和任务分配，确保了多个 AI 之间的顺畅协作。最终，通过自动化测试和人工验收，确保了网站的质量和稳定性。

## GitHub Pages 部署与访问
本项目已部署到 GitHub Pages，可以通过以下入口访问：
- 入口地址：`index.html`
- 访问方式：在浏览器中输入 `https://<用户名>.github.io/<仓库名>` 即可访问。

请将 `<用户名>` 和 `<仓库名>` 替换为实际的 GitHub 用户名和仓库名。通过 GitHub Pages 部署，实现了网站的静态文件托管和便捷访问。
